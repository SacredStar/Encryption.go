Данная библиотека предоставляет доступ к криптографическим функциям win32 с уклоном в работу с КриптоПро CSP 5.0 R2.
Основное направление работы составление zero-dependency  пакета для выполнения низкоуровневых криптографических операций и разработку высокоуровнего пакета для более нативной(go-style) Go  разработки.
В связи с тем что встраивание КриптоПро CSP  возможно только при использовании функций перечисленных в приложении 2 Правил пользования на CSP,
уклон при разработке библиотеки сделан в поддержку этих функций.
