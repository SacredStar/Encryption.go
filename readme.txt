Данная библиотека предоставляет доступ к криптографическим функциям win32 с уклоном в работу с КриптоПро CSP 5.0 R2.
Основное направление работы составление zero-dependency  пакета для выполнения низкоуровневых криптографических операций и разработку высокоуровнего пакета для более нативной(go-style) Go  разработки.
В связи с тем что встраивание КриптоПро CSP  возможно только при использовании функций перечисленных в приложении 2 Правил пользования на CSP, уклон при разработке библиотеки сделан в поддержку этих функций.

Все функциональные файлы расположены в директории GostCrypto.
GOST.go - высокоуровневая реализация криптографических операций
Win32GoFunctions -  низкоуровневые вызовы win32, для удобства в аннотации функции указаны win32 сигнатуры и краткое описание
Examples -  предоставляет примеры использования низкоуровневых функций, примеры идеалогически соответствуют примерам из SDK КриптоПро CSP 5.0 R2 (файл CSP.chm)

Тестирование проводится навстречу сертифицированым СКЗИ. В частности, планируется использовать: cpverify (из состава КриптоПро CSP),КриптоАрм ГОСТ, утилиты КриптоПроCSP
